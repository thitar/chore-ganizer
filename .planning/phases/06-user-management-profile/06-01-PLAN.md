    - Test: creates user with hashed password

    - Test: throws 400 on invalid email format

    - Throws 400 if id === requestingUserId (self-delete)
    - Deletes user

    - Test: throws 404 on non-existent

    - Updates with hashed new password
    - Throws 401 on wrong current password
    - Throws 400 on new password < 6 chars
    - Test: updates with correct current
    - Test: throws 401 on wrong current
    - Test: throws 400 on short new

    - Updates color
    - Validates hex format
    - Test: updates color
    - Test: throws 400 on invalid hex
  </behavior>
