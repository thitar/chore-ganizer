    - User select (conditionally rendered when `showUserFilter` is true): `<select value={userFilter} onChange={e => onUserChange?.(e.target.value)} className="...">`. Options: "All Users" (value="all") + mapped `users` array.

    - Clear button: `<button onClick={onClear} className="text-sm text-gray-500 hover:text-gray-700">Clear filters</button>`.
    - Copywriting per UI-SPEC: "Filter by status", "Filter by user" (labels not displayed as text — they're implicit from the select options/placeholders. The clear button label is "Clear filters").

    - Props: `{ status: 'PENDING' | 'COMPLETED'; overdue?: boolean }`.
    - Pending badge: `<span className="inline-flex items-center px-3 py-1 rounded-full text-xs font-normal bg-yellow-100 text-yellow-800">Pending</span>`.
    - Completed badge: `<span className="inline-flex items-center px-3 py-1 rounded-full text-xs font-normal bg-green-100 text-green-800">Completed</span>`.

    - Props: `{ message: string; deleteLabel: string; keepLabel: string; onDelete: () => void; onCancel: () => void; isDeleting?: boolean }`.
    - Renders inline red confirmation panel: `<div className="bg-red-50 border border-red-200 rounded-lg p-4">`.

    - Keep button: `<button onClick={onCancel} disabled={isDeleting} className="bg-white border border-gray-300 text-gray-700 px-3 py-2 rounded-lg text-sm hover:bg-gray-50 disabled:opacity-50">{keepLabel}</button>`.
