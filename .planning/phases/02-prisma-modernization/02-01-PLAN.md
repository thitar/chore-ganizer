  <action>
    Replace the `prisma.$use(async (params, next) => { ... })` middleware block (lines 15-47) with a `$extends({ query: { recurringChore: { ... } } })` call.

    Current `$use` middleware handles:
    - **Serialize (write):** `create`, `update`, `upsert`, `createMany`, `updateMany` — if `data.recurrenceRule` is an object, JSON.stringify it
    - **Upsert sub-args:** For upsert, serialize on BOTH `create` and `update` sub-objects (lines 27-34)
    - **Deserialize (read):** `findMany`, `findUnique`, `findFirst` — if result `recurrenceRule` is a string, JSON.parse it
    - **Array vs single handling:** Normalize `data` to array, process each item, return to original shape

    **Implementation:**

    Replace the entire `prisma.$use(...)` call with:

    ```typescript
    export const prisma =
      globalForPrisma.prisma ||
      new PrismaClient({
        log: process.env.NODE_ENV === 'development' ? ['query', 'error', 'warn'] : ['error'],
      })

    const xprisma = prisma.$extends({
      query: {
        recurringChore: {
          // -- WRITE handlers: serialize recurrenceRule object → JSON string --
          async create({ args, query }) {
            if (args.data?.recurrenceRule && typeof args.data.recurrenceRule === 'object') {
              args.data.recurrenceRule = JSON.stringify(args.data.recurrenceRule)
            }
            return query(args)
          },
          async update({ args, query }) {
            if (args.data?.recurrenceRule && typeof args.data.recurrenceRule === 'object') {
              args.data.recurrenceRule = JSON.stringify(args.data.recurrenceRule)
            }
            return query(args)
          },
          async upsert({ args, query }) {
            if (args.create?.recurrenceRule && typeof args.create.recurrenceRule === 'object') {
              args.create.recurrenceRule = JSON.stringify(args.create.recurrenceRule)
            }
            if (args.update?.recurrenceRule && typeof args.update.recurrenceRule === 'object') {
              args.update.recurrenceRule = JSON.stringify(args.update.recurrenceRule)
            }
            return query(args)
          },
          async createMany({ args, query }) {
            if (args.data) {
              const items = Array.isArray(args.data) ? args.data : [args.data]
              for (const item of items) {
                if (item?.recurrenceRule && typeof item.recurrenceRule === 'object') {
                  item.recurrenceRule = JSON.stringify(item.recurrenceRule)
                }
              }
            }
            return query(args)
          },
          async updateMany({ args, query }) {
            if (args.data?.recurrenceRule && typeof args.data.recurrenceRule === 'object') {
              args.data.recurrenceRule = JSON.stringify(args.data.recurrenceRule)
            }
            return query(args)
          },
          // -- READ handlers: deserialize recurrenceRule JSON string → object --
          async findMany({ args, query }) {
            const result = await query(args)
            if (Array.isArray(result)) {
              for (const item of result) {
                if (item?.recurrenceRule && typeof item.recurrenceRule === 'string') {
                  item.recurrenceRule = JSON.parse(item.recurrenceRule)
                }
              }
            }
            return result
          },
          async findUnique({ args, query }) {
            const result = await query(args)
            if (result?.recurrenceRule && typeof result.recurrenceRule === 'string') {
              result.recurrenceRule = JSON.parse(result.recurrenceRule)
            }
            return result
          },
          async findFirst({ args, query }) {
            const result = await query(args)
            if (result?.recurrenceRule && typeof result.recurrenceRule === 'string') {
              result.recurrenceRule = JSON.parse(result.recurrenceRule)
            }
            return result
          },
          async findFirstOrThrow({ args, query }) {
            const result = await query(args)
            if (result?.recurrenceRule && typeof result.recurrenceRule === 'string') {
              result.recurrenceRule = JSON.parse(result.recurrenceRule)
            }
            return result
          },
          async findUniqueOrThrow({ args, query }) {
            const result = await query(args)
            if (result?.recurrenceRule && typeof result.recurrenceRule === 'string') {
              result.recurrenceRule = JSON.parse(result.recurrenceRule)
            }
            return result
          },
        },
      },
    })
    ```

    **Key changes:**
    - Remove `export` from `prisma` declaration — it becomes `export default xprisma` as the primary export for the rest of app
    - Export `xprisma` as the default and named export (replace all `export const prisma` with `export default xprisma` and `export { xprisma as prisma }`)
    - The `globalForPrisma` guard remains for dev hot-reload — assign `xprisma` to `globalForPrisma.prisma` instead of `prisma`
    - Keep the `globalThis` guard: `if (process.env.NODE_ENV !== 'production') { globalForPrisma.prisma = xprisma }`

    Wait, there's a subtlety — Prisma `$extends` returns a new extended client type (`PrismaClient extends { ... }`). This means:
    - The export type changes from `PrismaClient` to the extended type
    - `xprisma` replaces `prisma` in all downstream imports
    - Downstream code that imports `prisma` from database.ts will get the extended client — no changes needed in services

    **Implementation detail:** The simplest approach is to keep `const xprisma = prisma.$extends(...)` and export it as:

    ```typescript
    export const prisma = xprisma
    export default xprisma
    ```

    This way all existing imports (`import { prisma } from './config/database'`) continue to work unchanged, while the `as any` cast at line 29 is eliminated since `$extends` args are fully typed.

    **Per D-03:** Single `$extends({ query: { ... } })` call — one extension covering both serialize and deserialize handlers.
    **Per D-01:** Use `$extends` query extension as direct `$use` replacement.
    **Per D-02:** Keep manual array vs. single-object argument handling (used in createMany handler).
  </action>
