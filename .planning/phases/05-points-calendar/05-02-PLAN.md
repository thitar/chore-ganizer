  <behavior>
    **Service: getMyPoints(userId)**
    - Returns `{ user, balance, logs[] }`
    - `balance = SUM(PointLog.amount WHERE userId = X)` (use Prisma aggregate)
    - `logs` = PointLog rows for user, ordered by createdAt desc, limit 100
    - `user` = User record with name, color
    - Throws 404 if user not found
    - Test: returns correct balance sum
    - Test: logs sorted by createdAt desc
    - Test: limit 100

    **Service: getUserPoints(userId, requestingUserId, requestingUserRole)**
    - Same as getMyPoints but for any user
    - If role is CHILD and userId !== requestingUserId → 403
    - Test: parent can access any user
    - Test: child can only access own

    **Service: adjustPoints(userId, amount, reason)**
    - Validates: amount is non-zero integer, reason is 1-200 chars
    - Creates PointLog with type='ADJUSTMENT', amount, reason
    - Returns the created log
    - Test: creates log with correct fields
    - Test: throws AppError 400 on invalid amount
    - Test: throws AppError 400 on empty reason
    - Test: throws AppError 404 if user not found
  </behavior>

  <implementation>
    1. Create `backend-v2/src/services/points.service.ts` with: `getMyPoints`, `getUserPoints`, `adjustPoints`
    2. Create `backend-v2/src/routes/points.routes.ts`:
       - `GET /me` — `authenticate` only
       - `GET /users/:id` — `authenticate` only, ownership check in service
       - `POST /adjust` — `authenticate` + `authorize('PARENT')`
    3. Mount in `routes/index.ts`: `router.use('/points', pointsRouter)`
  </implementation>
</feature>
