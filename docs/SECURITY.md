# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please report it responsibly.

### How to Report

1. **Email**: Send details to security@chore-ganizer.example.com
2. **Response Time**: We aim to respond within 48 hours
3. **Process**: 
   - We will acknowledge receipt of your report
   - We will investigate and confirm the vulnerability
   - We will develop and test a fix
   - We will release a patch and credit you (if desired)

### What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

### Safe Harbor

We support responsible disclosure. We will not take legal action against researchers who:
- Act in good faith
- Do not access or modify user data
- Do not degrade system performance
- Give us reasonable time to respond

Thank you for helping keep Chore-Ganizer secure!
